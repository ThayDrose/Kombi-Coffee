#  Kombi Coffee Landing Page

##  Visão Geral do Projeto e Contexto Acadêmico

Este projeto Front-End (Landing Page estática) foi desenvolvido como entrega obrigatória do **1º Trimestre** da graduação em Engenharia de Software no **Instituto Infnet**.

A complexidade e o rigor técnico exigidos já no início do curso demonstram a seriedade da formação, atestando a capacidade de desenvolvimento e entrega de código de alta qualidade desde as etapas iniciais.

##  Destaques Técnicos e Desafio

O objetivo central foi garantir a fidelidade de design e a performance técnica em um ambiente de alta cobrança.

* Estrutura e Semântica: HTML5 estruturado e acessível, com foco em otimização (SEO).
* Responsividade Sólida: Aplicação robusta de CSS Flexbox e *Media Queries* para garantir um layout 100% adaptativo em todos os dispositivos (*Mobile First*).
* Branding Visual: Simulação precisa da identidade visual (paleta de cores e tipografia) de um negócio real
